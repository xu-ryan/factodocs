## 1. 顶部栏设置

要启用顶部栏，请前往 `Appearance` **(外观)** > `Customize` **(自定义)** > `Header` > `Top Bar`。

<img
  style={{ borderRadius: '0.5rem' }}
  src="/images/top-bar-1.jpg"
/>

## 2. 顶部栏颜色

顶部栏使用的颜色可以通过 `Appearance` **(外观)** > `Customize` **(自定义)** > `Header` > `Top Bar` 进行更改。可以分别控制背景颜色和顶部栏的文本颜色。文本颜色也会应用于顶部栏的社交媒体图标。

<img
  style={{ borderRadius: '0.5rem' }}
  src="/images/top-bar-2.jpg"
/>

## 3. 顶部栏文本

要更改顶部栏中的文本，请前往 `Appearance` **(外观)** > `Customize` **(自定义)** > `Header` > `Top Bar`，或直接点击定制器预览中的编辑图标。

<img
  style={{ borderRadius: '0.5rem' }}
  src="/images/top-bar-3.jpg"
/>

## 4. 顶部栏导航

### 4.1 顶部栏菜单设置

通过 `Appearance` **(外观)** > `Menus` **(菜单)** 创建一个新菜单或将现有菜单分配到顶部栏的菜单位置。

<img
  style={{ borderRadius: '0.5rem' }}
  src="/images/top-bar-4.jpg"
/>

### 4.2 顶部栏菜单位置

顶部栏菜单可以显示在屏幕左侧，与文本一起，或显示在屏幕右侧，与社交媒体图标一起。要更改其位置，请前往 `Appearance` **(外观)** > `Customize` **(自定义)** > `Header` > `Top Bar`。

<img
  style={{ borderRadius: '0.5rem' }}
  src="/images/top-bar-5.jpg"
/>

## 5. 顶部栏社交媒体图标

顶部栏中的社交媒体图标可以通过 `Appearance` **(外观)** > `Customize` **(自定义)** > `Header` > `Top Bar` 启用或禁用。

<img
  style={{ borderRadius: '0.5rem' }}
  src="/images/top-bar-6.jpg"
/>

社交媒体图标和链接可以通过 `Appearance` **(外观)** > `Customize` **(自定义)** > `Social Media` 进行编辑。

<img
  style={{ borderRadius: '0.5rem' }}
  src="/images/top-bar-7.jpg"
