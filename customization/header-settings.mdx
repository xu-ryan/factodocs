## 1. 页眉布局

FactoShop 提供了几种不同的页眉布局。每种布局会以不同方式排列现有元素。要更改页眉布局，请前往 `Appearance` **(外观)** > `Customize` **(自定义)** > `Header` > `Header Styles`。

<img
  style={{ borderRadius: '0.5rem' }}
  src="/images/header-settings-1.jpg"
/>

## 2. 页眉大小

### 2.1 页眉宽度（盒装/全宽）

页眉的宽度可以设置为全宽或自定义宽度（以像素为单位）。要调整这些值，请前往 `Appearance` **(外观)** > `Customize` **(自定义)** > `Header` > `Header Styles`。

<img
  style={{ borderRadius: '0.5rem' }}
  src="/images/header-settings-2.jpg"
/>

### 2.2 页眉高度

可以通过调整 Logo 上下的间距来调整页眉的高度。要调整这些值，请前往 `Appearance` **(外观)** > `Customize` **(自定义)** > `Header` > `Logo`。

<img
  style={{ borderRadius: '0.5rem' }}
  src="/images/header-settings-3.jpg"
/>

## 3. 页眉颜色

### 3.1 导航字体颜色

要调整导航菜单使用的颜色，请前往 `Appearance` **(外观)** > `Customize` **(自定义)** > `Header` > `Header Colors`。

<img
  style={{ borderRadius: '0.5rem' }}
  src="/images/header-settings-4.jpg"
/>

### 3.2 页眉背景颜色

要更改页眉使用的背景颜色，请前往 `Appearance` **(外观)** > `Customize` **(自定义)** > `Header` > `Header Colors`。

<img
  style={{ borderRadius: '0.5rem' }}
  src="/images/header-settings-5.jpg"
/>

### 3.3 页眉背景图片

除了设置背景颜色外，页眉还支持背景图片。要为页眉上传背景图片，请前往 `Appearance` **(外观)** > `Customize` **(自定义)** > `Header` > `Header Styles`。

<img
  style={{ borderRadius: '0.5rem' }}
  src="/images/header-settings-6.jpg"
/>

## 4. 页眉常见问题


### 4.1 为什么选择居中 Logo 布局后看不到我的菜单？

一旦你使用两种居中 Logo 布局中的一种，并且菜单的一部分在左侧，一部分在右侧，你需要知道这种布局需要构建两个菜单。一个显示在左侧，一个显示在右侧。

创建这两个菜单后，确保为每个菜单选择红色标记的选项：
