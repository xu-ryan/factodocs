## 1. 固定页眉设置

### 1.1 启用/禁用固定页眉

固定页眉的设置可以在 `Appearance` **(外观)** > `Customize` **(自定义)** > `Header` > `Sticky Header` 中找到。在这里，你可以启用或禁用它，自定义其颜色，并调整大小和间距。

<img
  style={{ borderRadius: '0.5rem' }}
  src="/images/sticky-header-1.jpg"
/>

### 1.2 包含/排除顶部栏

顶部栏可以包含在固定页眉中。该选项可以在 `Appearance` **(外观)** > `Customize` **(自定义)** > `Header` > `Sticky Header` 中找到。

<img
  style={{ borderRadius: '0.5rem' }}
  src="/images/sticky-header-2.jpg"
/>

## 2. 固定页眉颜色

固定页眉可以有自己的配色方案，这里的自定义设置将允许你改变背景颜色和文本颜色。

<img
  style={{ borderRadius: '0.5rem' }}
  src="/images/sticky-header-3.jpg"
/>

## 3. 固定页眉 Logo

固定页眉使用不同的配色方案可能需要一个替代徽标 (例如在浅色背景上使用深色 Logo)。在这里上传的图像将仅显示在固定页眉上。

<img
  style={{ borderRadius: '0.5rem' }}
  src="/images/sticky-header-4.jpg"
/>

## 4. 固定页眉大小

固定页眉的大小可以通过调整固定页眉 Logo 的大小以及 Logo 上下间距来自定义。

<img
  style={{ borderRadius: '0.5rem' }}
  src="/images/sticky-header-5.jpg"
/>

## 5. 移动端固定页眉

移动端页眉也可以通过 `Appearance` **(外观)** > `Customize` **(自定义)** > `Header` > `Mobile Header` 设置为固定。
